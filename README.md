## 手順

- ローカルにフォルダ作成し、ここのコードをクローン

```bash
% mkdir RhythmTraningApp
% cd RhythmTraningApp
% git clone https://github.com/motoshifurugen/RhythmTrainingApplication.git
```

- 実行

```bash
% cd RhythmTrainingApplication
% npm install
% npm run dev
```

- ブラウザで確認

http://localhost:8080/
